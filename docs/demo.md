## 3. 制作 GIF（推荐用 OBS）

### 录制流程（约 60 秒）

```
0-10s:  打开首页，输入"告别诗"，展示搜索结果
10-20s: 滚动查看识别到的角色列表
20-35s: 填写用户画像（MBTI INFP / 星座天蝎 / 女）
35-45s: 回答系统生成的追问
45-55s: 展示匹配推荐结果 + 理由
55-60s: 展示角色对比功能
```

### OBS 设置建议

| 参数 | 建议值 |
|------|--------|
| 分辨率 | 1280×720 |
| 帧率 | 15 FPS（文档展示用，不需要 30fps） |
| 格式 | MP4 → 转 GIF |
| 转 GIF 工具 | [ezgif.com](https://ezgif.com/video-to-gif) |
